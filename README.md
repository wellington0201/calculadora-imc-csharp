# 📊 Calculadora de IMC em C#

Aplicação desenvolvida em Windows Forms para cálculo do IMC (Índice de Massa Corporal).

---

## 🚀 Funcionalidades

- Inserção de peso e altura
- Validação de dados com TryParse
- Cálculo automático do IMC
- Classificação do resultado:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade Grau I
  - Obesidade Grau II
  - Obesidade Grau III

---

## 🛠 Tecnologias utilizadas

- C#
- .NET
- Windows Forms

---

## 📚 Conceitos aplicados

- Estrutura condicional (if/else)
- Validação de entrada de dados
- Eventos em Windows Forms
- Manipulação de interface gráfica

---

## ▶️ Como executar o projeto

1. Abrir o projeto no Visual Studio
2. Executar (F5)
3. Inserir peso e altura
4. Clicar no botão "Calcular"
